
## House Price Prediction

## Task Objective
The objective of this task is to predict house prices using machine learning regression models based on property features such as area, number of rooms, and location.

## Dataset Used
The dataset used is the Amsterdam House Price Prediction Dataset from Kaggle.
It includes the following features:
Area – size of the house (m²)
Room – number of rooms
Lon – longitude
Lat – latitude
Price – target variable

## Models Applied
The following regression models were implemented:
Linear Regression
Gradient Boosting Regressor

## Key Results and Findings
Gradient Boosting performed better than Linear Regression
It achieved lower error values (MAE and RMSE)
House prices are strongly influenced by area and location
Non-linear models provide better predictions for real estate data
